# Groups Strategy: Roles And Permissions

## Managing Roles

Default `groups.manage_roles`:

- `president`
- `delegate`
- `alternate_delegate`
- `council_delegate`
- `council_alternate_delegate`
- `correspondent`

## Roster Roles

Default `groups.roster_roles`:

- `member`
- `observer`

Only actors with an allowed managing role can use group mutation actions.
